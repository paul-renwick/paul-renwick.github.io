What is the difference between Margin, Border, and Padding?

Margin, border and padding all refer to different areas of space associated with an element. In order to best visualize this I have created an image below which illustrates their respective locations of space when surrounding content.

For the more technically inclined, it is worth noting that when dabbling with margins, borders and padding, you are able to change the size of each side individually, rather than just the whole area as a whole. When tinkering with these values, CSS assigns values in the order of top, right, bottom, left. For example, if I was to adjust the padding of a div in css, I would use the following semantic structure.

div {

padding: 30px 100px 80px 20px

}

This would create a padding with 30px on top, 100px on the right side, 80px on the bottom and 20px on the left side.
