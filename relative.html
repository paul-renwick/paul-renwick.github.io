What are the differences between relative, absolute, and fixed positioning?

For the sake of avoiding plagiarism, it should be noted that a lot of my knowledge on this subject came from here

In order to best understand relative, absolute and fixed positioning it is useful to first understand that everything on a webpage takes place within a box.

Relative positioning effects how an element is positioned relative to itself, this means that if you set an element to position: relative; it would have no effect. In order for relative positioning to have an effect you must assign a value to where it will be positioned, for example, if you set bottom: 10px, then the elements position would be 10px above where it would normally be.

Absolute positioning allows an element to be placed anywhere on the page. An element given this value will be fixed to the location assigned to it regardless of other elements. An element is placed using absolute positioning assigning it a top, left, bottom and right attribute.

Fixed positioning fixes an element to a position on the browser window, the element will not move when the user scrolls through the browser, you could perhaps use this to fix a navigation pane to the side or top of a screen.
