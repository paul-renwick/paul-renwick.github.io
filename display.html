What does it mean to display inline vs inline blocks?

Display inline and display inline blocks refer to css values which effect how an element is displayed.

When an element is given the inline property, the element will align in a horizontal line. When an element is given the inline-block value, it will also display in a line, but it may now be assigned a width and height value. Another similar value is display: block, this differs from inline-block in that it adds a line break so it will sit above or below the surrounding elements.
